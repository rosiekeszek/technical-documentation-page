What Is HTML?

HTML stands for Hypertext Markup Language. It is a system of symbols or codes which, when inserted into a file, tells web browsers how to display a web page's text and images. These pieces of code are called tags or elements.

HTML Tags

HTML tags are generally used in pairs, one tag is placed at the beginning of the information (text or image to be displayed on the web page), another is placed at the end of the information. These are called opening tags and closing tags. The tags function is to tell the web browser what to do with the content in between them.

An opening tag will look like this:

<header>
with no spaces between the brackets and the characters.

A closing tag will contain a slash like this:

</header>
Between these two tags would be the content which appears on the screen. Here's an example:
<header>This is a header</header>
Note that there are no spaces between the brackets and the text. The words 'This is a header' are all that you will see on the web page.

Attributes and Values

Opening tags may have attributes. Attributes must have a name and a value. Values are important for applying CSS rules to your HTML. This is an example of an opening tag with attributes and values:

<header id="main-header" class="header-1">

In this example the id and class are attributes assigned to this header tag. Each has a value which is in double quotation marks. Attributes are used to identify sections of HTML so that CSS (Cascading Style Sheet) rules an be applied to the content between the tags. Attributes, like tags, are specific codes whereas values can be made up by the coder.

Empty Elements

There are a few elements which consists of a single tag rather than the opening and closing paired tags. These are called empty elements. Examples of empty elements are:

  • <br>
  • <hr>

The first element in this list is called a break tag and is used to create a break in the content, whatever comes after this element will start on a new line. The second element is used to a horizontal line which is useful in separating sections with different topics.

Semantic Markup

Semantic markup elements are used to indicate emphasis in a text rather than define the structure of the page. Some examples of semantic markup are:

  • <strong>
  • <em>
  • <ins>
  • <del>
  • <s>

These tags allow us to note where text should be emphasized by making the font bold or by using italics. The last three tags are used to show changes to the text. Like when you want the viewer reader to see that you changed your mind about which words to use. In the last sentence I used the del and ins tags to show which word I would have deleted and which word I would have chosen to insert to replace it. The last tag shows another way to do this. The s tag strikes through information that you would like to remain visible. Such as original price vs sale price on an advertisement.

Creating Lists in HTML

There are three types of lists in HTML:

  • ordered lists
  • unordered lists
  • definition lists

These are pretty straightforward. Ordered lists will arrange the items in ascending order useing numbers or letters, unordered lists will insert bullet points, definition lists create a format which shows a definition, such as a dictionary entry
